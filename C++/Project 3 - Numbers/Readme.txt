Project 3 - Numbers
This project is to create an array and fill it using a file. Once the values from the file are inserted 
into the array we will display the values of the array. The program will then calculate the minimum, 
maximum, sum, and average. All of these values will then be displayed and the program will close.


PROTOTYPE

void Show_Array();
int Read_From_User_File();
void Calc_Stats();
